## 🚀 Destaques de Arquitetura e Engenharia de Defesa

### 👁️ [Observabilidade SecOps Unificada (K8s + Wazuh + Grafana)](https://github.com/kaxcav0tracer/k8s-wazuh-lab)
* **A Arquitetura:** Engenharia de um *Single Pane of Glass* correlacionando métricas de performance (Zabbix) e eventos de segurança (Wazuh) dentro de um cluster Kubernetes.
* **O Impacto:** Quebra de silos de dados entre DevOps e Blue Team, permitindo a correlação instantânea entre anomalias de consumo de recursos (CPU/RAM) e alertas de invasão.
* **Stack:** `Kubernetes`, `Wazuh`, `Grafana`, `Zabbix`, `OpenSearch`.

### ⚡ [Cloud SIEM & SecOps Automation Suite](https://github.com/kaxcav0tracer/secops-automation-suite-v2)
* **A Arquitetura:** Provisionamento de um laboratório SOC híbrido ancorado no Microsoft Sentinel, integrado a uma suíte de automação customizada.
* **O Impacto:** Desenvolvimento de 6 ferramentas de automação em PowerShell/Python (com interface gráfica WPF) para auditoria de Active Directory, hunting com KQL e integração com APIs de Threat Intelligence.
* **Stack:** `Microsoft Sentinel`, `KQL`, `PowerShell (GUI)`, `Python`, `Azure Cloud`.

### 🛡️ Secure Gateway & Endpoint Hardening
* **A Arquitetura:** Substituição de exposição direta de portas RDP/SSH por um gateway de acesso remoto *clientless* (Apache Guacamole) e implementação massiva de Windows LAPS via GPO.
* **O Impacto:** Redução drástica da superfície de ataque do perímetro da rede governamental e mitigação efetiva de movimentos laterais (Pass-the-Hash) em ambientes Active Directory.
* **Stack:** `Apache Guacamole`, `Linux Hardening`, `Windows Server (LAPS)`, `Active Directory`.
